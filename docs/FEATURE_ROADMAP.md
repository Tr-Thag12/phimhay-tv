# Roadmap phát triển PhimHay TV

## Giai đoạn 1: Hoàn thiện frontend mock

Trạng thái: đã xong.

Các hạng mục đã hoàn thành:

- Chuẩn hóa project sang Vite.
- Tách frontend thành module.
- Thiết kế lại giao diện.
- Đánh bóng UI.
- Xử lý fallback ảnh.
- Thêm URL routing thật bằng History API.
- Thêm SEO cơ bản.
- Chuẩn bị cấu hình deploy Vercel.
- Deploy demo online.

## Giai đoạn 2: Chuẩn hóa dữ liệu và API giả

- Chuẩn hóa model phim.
- Chuẩn hóa model tập phim.
- Chuẩn hóa model thể loại.
- Tạo mock service thay cho import data trực tiếp.
- Chuẩn bị chuyển sang API thật.

## Giai đoạn 3: Backend cơ bản

- Chọn stack backend đề xuất.
- Tạo server API.
- Tạo database.
- CRUD phim.
- CRUD thể loại.
- CRUD tập phim.
- API tìm kiếm/lọc phim.
- API chi tiết phim.
- API tăng lượt xem.

## Giai đoạn 4: Admin

- Admin login.
- Dashboard.
- Quản lý phim.
- Quản lý tập phim.
- Quản lý thể loại.
- Quản lý người dùng.
- Quản lý bình luận.

## Giai đoạn 5: User account

- Đăng ký.
- Đăng nhập.
- Đăng xuất.
- Hồ sơ cá nhân.
- Watchlist theo tài khoản.
- History theo tài khoản.
- Tiếp tục xem.

## Giai đoạn 6: Tương tác

- Đánh giá phim.
- Bình luận phim.

- Gợi ý phim liên quan.
- Top phim xem nhiều.

## Giai đoạn 7: Tối ưu
