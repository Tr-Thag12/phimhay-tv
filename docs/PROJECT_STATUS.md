# Trạng thái dự án PhimHay TV

## Thông tin chung

- Tên dự án: PhimHay TV.
- Repo GitHub: https://github.com/Tr-Thag12/phimhay-tv
- Demo Vercel: https://phimhay-tv.vercel.app/
- Mô tả ngắn: PhimHay TV là frontend mock cho website xem phim, có giao diện dark cinematic, dữ liệu phim mẫu, định tuyến URL thật và deploy demo trên Vercel.

## Công nghệ hiện tại

- Vite.
- HTML/CSS/JavaScript module.
- `localStorage` cho watchlist và history mock.
- History API routing.
- Vercel deploy.

## Các phần đã hoàn thành

- Chuẩn hóa project sang Vite.
- Tách frontend thành module.
- Redesign UI theo hướng dark cinematic.
- Xử lý fallback ảnh phim.
- Thêm URL routing thật bằng History API.
- Thêm SEO cơ bản gồm title, meta description và canonical.
- Chuẩn bị cấu hình deploy Vercel.
- Deploy demo Vercel và cập nhật link demo vào tài liệu.
